# TVK 2026 Case Analysis

A GitHub Pages-ready editorial website for Hemanth Lal's strategic analysis of TVK's 2026 Tamil Nadu election win.

## Publish on GitHub Pages

1. Push this folder to a GitHub repository.
2. Open `Settings > Pages`.
3. Select `Deploy from a branch`.
4. Choose the repository branch and `/ (root)`.

The site is plain HTML, CSS, and JavaScript. It does not require a build step.

## Image sources

The Vijay portrait and campaign images are downloaded from the official [TVK website](https://tvkvijay.com/en) and [TVK gallery](https://tvkvijay.com/en/gallery/election-campaign-in-tiruppur). Image credits are included in the site captions.
